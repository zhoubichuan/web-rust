## 1.宏

### 1.1 `println!`

```rust
println!("hello");
```

#### 1.1.1 基本占位

```rust
let x = 1;
let y = 2;
println!("x={},y={}",x,y);
```

#### 1.1.2 下标占位

```rust
let x = 1;
let y = 2;
println!("x={0},y={1}",x,y);
```

#### 1.1.3 命名参数

```rust
let x=  1;
let y = 2;
println!("x={x},y={y}");
println!("x={x},y={y}",x=5,y=6);
```

#### 1.1.4 函数参数

```rust
fn string_fn(a:&str,b:&str)->String{
format!("{b},{a}")
}
println!("{}",string_fn("ABC","EFG"));
```

#### 1.1.5 宽度

```rust
println!("Hello {:5}!", "x");
```
