# 四.数据类型

::: tip 前言
rust 数据类型分为：

- 1.标量类型
- 2.复合类型
  :::

## 1.标量类型

### 1.1 整型

整型默认类型为`i32`

#### 1.1.1 有符号整数

`i`前缀

```rust
let number: i32 = 42;
```

#### 1.1.2 无符号整数

`u`前缀

```rust
let number: u64 = 100
```

#### 1.1.3 类型转换

```rust
assert_eq!(10_i8 as u16,10_u16)
```

- 截断
  数据从大数转位小数可能会被截断

- Checked
- Wrapping
- Staturating
- Overflowing

### 1.2 浮点数

浮点数默认类型为`f64`

#### 1.2.1 单精度浮点数

```rust
let float_number: f32 =3.14
```

#### 1.2.2 双精度浮点数

```rust
let double_number: f64 = 3.1415926
```

### 1.3 字符

- 字符类型使用单引号`'`表示

```rust
let word:char = 'A'
```

### 1.4 布尔值

- 值为`true`或`false`

```rust
let value:bool = true
```

## 2.复合数据类型

### 2.1 数组

- 数组固定大小，且所有元素类型必须相同，使用`[]`表示

```rust
let arr:[i32; 3] = [1,2,3]
```

### 2.2 元祖

- 存储多个不同类型的值，使用`()`表示

```rust
let tuple:(i32,f64,char)=(42,3.14,'A')
```

### 2.3 切片

- 引用数组或向量的一部分数据，使用`&[T]`表示

```rust
let arr:[i32;5]=[1,2,3,4,5]
let slice:&[i32]=&arr[...3] // 引用数组
```

### 2.4 字符串

字符串包含：`容量`

#### 2.4.1 创建

- 1.`字面量`方式

可以不用指定类型，rust 会自动推导

```rust
let a = "aaa";
// let a: $str= "aaa"; // 指定类型
```

- 2.`new`

创建一个空的 String 对象

```rust
let string= String::new();//""
```

- 3.`from`

从一个字符串字面量、一个字节数组或另一个字符串对象中创建一个新的 String 对象

```rust
let s1 :String = String::from("hello"); // "hello"
let s2:String = String::from_utf8(vec![1,3,43,5,6]).unwrap()
let s3:String= String::from(s1)
```

- 4.`witch_capacity`

创建一个具有指定容量的 String 对象

```rust
let mut s= String::witch_capacity(10);
s.push("a")
```

- 5.`capacity`

返回字符串的容量（已字节为单位）

```rust
let mut s= String::witch_capacity(10);
assert_eq!(s.capacity(),10)
```

- 6.`clone`

创建一个与原始字符串相同的新字符串

```rust
let s1 = String::from("hello");
let s2 = s1.clone();
```

#### 2.4.2 添加

- 1.`push`

将一个字符追加到字符串末尾

```rust
let mut s = String::from("abc");
s.push('!') ;//追加字符 -> "abcobb!"
```

- 2.`push_str`

将一个字符串追加到字符串末尾

```rust
let mut s = String::from("abc");
s.push_str("obb"); // 追加字符串切片
```

#### 2.4.3 删除

- `pop`

将字符串的最后一个字符弹出，并返回他

```rust
let mut s = String::from("hello");
let last = s.pop();
```

#### 2.4.4 分割

- `split`

将字符串分割为一个迭代器，每个元素都是一个子字符串

```rust
let s = String::from("hello,world");
let mut iter=s.split(",");
assert_eq!(iter.next(),Some("hello"))
assert_eq!(iter.next(),Some("world"));
assert_eq!(iter.next(),None)
```

#### 2.4.5 查询

- `len`

返回字符串的长度（以字节为单位）

```rust
let s = "你好";
let len = s.len(); // 值为6，长度算的是编码长度
```

- `count`

```rust
let s = String::from("heasd中国");
let len = s.chars().count();//7
```

- `chars`

将字符串分割为一个迭代器，每个元素都是一个字符

```rust
let s = String::from("heasd中国");
for c in s.chars() {
    println!("数据是: {c}");
}
// 数据是: h
// 数据是: e
// 数据是: a
// 数据是: s
// 数据是: d
// 数据是: 中
// 数据是: 国
```

- `is_empty`

检查字符串是否为空

```rust
let s = String::from("");
assert!(s.is_empty());
```

- `eq`

比较两个字符串是否相等

```rust
let s1 = String::from("hello");
let s2 = Stirng::from("hello");
assert!(s1.eq(s2));
```

- `contains`

检查字符串是否包含指定的子字符串

```rust
let s = Stirng::from("hello");
assert!(s.contents("e"));
```

- `starts_with`

检查字符串是否以指定的前缀开头

```rust
let s = String::from("hello");
assert!(s.starts_width("he"));
```

- `ends_with`

检查字符串是否以指定后缀结尾

```rust
let s = String::from("hello");
assert!(s.ends_with("lo"));
```

- `find`

查找字符串中第一个匹配指定字符串的位置

```rust
let s = String::from("hello");
let pos = s.find("1");
assert_eq!(pos,Some(2));
```

#### 2.4.6 类型转换

- `to_string`

将其他类型转为字符串

```rust
let one = 1.to_string() //  整数转字符串
let float = 1.3.to_string() // 浮点数转字符串
let slice = "slice".to_string(); // 字符串切片转字符串
```

- `as_bytes`

将 String 对象转换为字节数组

```rust
let s = String::from("hello");
let bytes = s.as_bytes();
```

- `into_bytes`

将 String 对象转换为字节向量

```rust
let s = String::from("hello");
let bytes = s.into_bytes();
assert_eq!(&[104,101][..],&bytes[..])
```

- `as_str`

将 String 对象转换为字符串切片

```rust
let s = String::from("hello");
let slice = a.as_str();
```

- `as_mut_str`

将 String 对象转换为可变字符串切片

```rust
let mut s = String::from("foobar");
let s_mut_str = s.as_mut_str()
s_mut_str.make_ascii_uppercase()
assert_eq!("FOOBAR",s_mut_str)
```

- `trim`

删除字符串两端的所有空格

```rust
let s = String::from("   hello  ");
let trim = s.trim();
```

- `to_lowercase`

将字符串中的所有字符转换为小写

```rust
let s = String::from("HeLLO");
let lower = s.to_lowsercase();
```

- `to_uppercase`

将字符串中的所有字符转换为大写

```rust
let s = String::from("HeLLo");
let upper = s.to_uppercase();
```

- `retain`

保留满足指定条件的所有字符，即筛选

```rust
let mut s = String::from("hello");
s.retain(|c| c!='1')
```

#### 2.4.7 字符串拼接

- `+`连接

```rust
let s1 = String::from("hello,");
let s2 = String::from("world!");
let s3 = s1 + &s2;//"hello,world!"
```

- format

```rust
let s1 = String::from("hello,")
let s2 = String::from("world!")
let s3 = format!("{}-{}",s1,s2)
```

## 3.自定义数据类型

### 3.1 结构体类型

- 名称采用大驼峰命名法，属性采用键值对命名法
- 定义复杂的数据结构，使用`struct`关键字定义

```rust
struct Person {
 name:String,
 age:u32,
}
let person = Person {
  name:String::from('Alice'),
  age:23
}
```

#### 3.1.1 名值对

#### 3.1.2 元祖式

#### 3.1.3 单元式

#### 3.1.4 增加方法

使用`impl`关键字

### 3.2 枚举类型

- 定义具有不同变体的类型，使用`enum`关键字定义

```rust
enum Color {
  Red,
  Green,
  Blue,
}
let color:Color = Color::Red;
```

### 3.3 引用类型

- 引用类型用于引用其他变量的值，而不是拥有自己的所有权，使用`&`符号表示

```rust
let val:i32 = 42
let refer: &i32=&value
```

#### 3.3.1 只读引用

用`&`表示

##### 3.3.1.1 共享访问

#### 3.3.2 可写引用

##### 3.3.2.1 排他访问

用`&mut`表示

#### 3.3.3 解引用

用`*`表示

### 3.4 向量类型

- 存储可变大小的元素序列，使用`Vec<T>`表示

```rust
let mut vector:Vec<i32> = Vec::new()
vector.push(1)
vector.push(2)
vector.push(3)
```

### 3.5 函数类型

- 函数类型由函数的参数类型和返回类型组件

```rust
fn add(a:i32,b:i32) -> i32 {
  a + b
}
```

### 3.6 范围类型

- 左闭右开

```rust
use std::ops::{Range, RangeInclusive}
let val = (1..5)
let val2 = Range{ start: 1, end: 5 }
```

- 全闭

```rust
use std::ops::{Range, RangeInclusive}
let val = (1..=5)
let val2 = RangeInclusive::new(1, 5)
```

### 3.7 映射表

#### 3.7.1 创建

- new

```rust
use std::collections::HashMap;

fn main(){
    let mut map = HashMap::new();
    map.insert("color","red");
    map.insert("size","111");
    println!("{}",map.get("color").unwrap())
}
```

#### 3.7.2 查询

- `get`

获取值

```rust
map.get("color")
```

- `contains_key`

检查哈希表中是否包含特定的键

```rust
obj.contains_key("target")
```

#### 3.7.3 添加

- `insert`

插入新的数据

```rust
use std::collections::HashMap;

fn main() {
    let mut map = HashMap::new();
    map.insert("color", "red");
    map.insert("size", "10 m^2");
    for p in map.iter() {
        println!("{:?}", p);
    }
}
```

- `entry`

```rust
map.entry("color").or_insert("red");
```

- `get_mut`

```rust
use std::collections::HashMap;

fn main() {
    let mut map = HashMap::new();
    map.insert(1, "a");

    if let Some(x) = map.get_mut(&1) {
        *x = "b";
    }
}
```
