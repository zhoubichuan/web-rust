# let

变量有两部分组成：值 、 类型

值可以分为：不可变变量、可变变量

类型可以分为不同的[数据类型](/1.base/1.js/4.variableobj)

- 关键字`let`用于声明`不可变变量`

```rust
let a = 123;
```

- 关键字`mut`用于声明`可变变量`

```rust
let mut a = 123;
a = 456;
```
