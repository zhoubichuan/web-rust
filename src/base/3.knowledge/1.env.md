# 一.猜数字

```rust
use std::io; // use 使用模块  :: 模块下的 方法

fn main(){
    println!("guess the number"); // ! 表示宏
    println!("pleass input your guess");

    let mut guess = String::new(); // :: 类型下的静态方法
    io::stdin()
        .read_line(&mut guess) // . 调用方法
        .expect("failed to read line"); // expect 错误处理
    println!("you guessed:{guess}")
}
```
