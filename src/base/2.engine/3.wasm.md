# 四.wasm

::: tip 前言
使用`wasm-pack`创建`WebAssembly`项目
:::

## 1.创建项目

```sh
cargo install wasm-pack
```

```sh
cargo new hello-wasm --lib
```

## 2.项目构建

```sh
wasm-pack build --target web
```
