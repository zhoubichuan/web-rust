# 七.异步编程

前言

js 代码从上往下执行，遇到setTimeout、setInterval等包裹的代码块会跳过，先将其他代码执行完，然后执行之前跳过的代码

  • 同步代码在执行栈中执行完毕后
  • 异步队列中的代码进入执行栈中执行
  • 异步队列会分为宏任务队列、微任务队列两种类型
  • 当有微任务队列时，会首先在执行栈中执行微任务队列代码，直至微任务队列没有代码（执行代码期间产生的异步任务也会分别进入宏任务队列和微任务队列），然后执行宏任务队列代码
