# 四.数据类型

前言

rust 数据类型分为：

  • 1.标量类型
  • 2.复合类型

# 1.标量类型

# 1.1 整数

# 1.1.1 有符号整数

i前缀

let number: i32 = 42
1

# 1.1.2 无符号整数

u前缀

let number: u64 = 100
1

# 1.1.3 类型转换

assert_eq!(10_i8 as u16,10_u16)
1

# 1.2.1 单精度浮点数

# 1.4 字符类型

  • 字符类型使用单引号'表示

# 2.复合类型

# 2.1 元祖类型

  • 存储多个不同类型的值，使用()表示

  • 数组固定大小，且所有元素类型必须相同，使用[]表示
