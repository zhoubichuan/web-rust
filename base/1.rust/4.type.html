# 四.数据类型

前言

rust 数据类型分为：

  • 1.标量类型
  • 2.复合类型

# 1.标量类型

# 1.1 整型

整型默认类型为i32

# 1.1.1 有符号整数

i前缀

let number: i32 = 42;
1

# 1.1.2 无符号整数

u前缀

let number: u64 = 100
1

# 1.1.3 类型转换

assert_eq!(10_i8 as u16,10_u16)
1

  • 截断 数据从大数转位小数可能会被截断

  • Checked

  • Wrapping

  • Staturating

  • Overflowing

# 1.2 浮点数

浮点数默认类型为f64

# 1.2.1 单精度浮点数

let float_number: f32 =3.14
1

# 1.2.2 双精度浮点数

let double_number: f64 = 3.1415926
1

# 1.3 字符

  • 字符类型使用单引号'表示
let word:char = 'A'
1

# 1.4 布尔值

  • 值为true或false
let value:bool = true
1

# 2.复合数据类型

# 2.1 数组

  • 数组固定大小，且所有元素类型必须相同，使用[]表示
let arr:[i32; 3] = [1,2,3]
1

# 2.2 元祖

  • 存储多个不同类型的值，使用()表示
let tuple:(i32,f64,char)=(42,3.14,'A')
1

# 2.3 切片

  • 引用数组或向量的一部分数据，使用&[T]表示
let arr:[i32;5]=[1,2,3,4,5]
let slice:&[i32]=&arr[...3] // 引用数组
1
2

# 2.4 字符串

字符串包含：容量

# 2.4.1 创建

  • 1.字面量方式

可以不用指定类型，rust 会自动推导

let a = "aaa";
// let a: $str= "aaa"; // 指定类型
1
2
  • 2.new

创建一个空的 String 对象

let string= String::new();//""
1
  • 3.from

从一个字符串字面量、一个字节数组或另一个字符串对象中创建一个新的 String 对象

let s1 :String = String::from("hello"); // "hello"
let s2:String = String::from_utf8(vec![1,3,43,5,6]).unwrap()
let s3:String= String::from(s1)
1
2
3
  • 4.witch_capacity

创建一个具有指定容量的 String 对象

let mut s= String::witch_capacity(10);
s.push("a")
1
2
  • 5.capacity

返回字符串的容量（已字节为单位）

let mut s= String::witch_capacity(10);
assert_eq!(s.capacity(),10)
1
2
  • 6.clone

创建一个与原始字符串相同的新字符串

let s1 = String::from("hello");
let s2 = s1.clone();
